This is a small game done for a 5-day game jam with adaptation as the theme. This was the first game jam which I utilized Unreal Engine for development after learning how to use the engine over the preceding months. It is meant to be a rogue-lite where the player's death increases certain stats as the player explores levels of a dungeon with increase difficulty per level. To help with fast iteration it was done fully in blueprint with all models, sounds, effects and the dungeon level being taken from the Unreal asset store. The gun system was taken from an online tutorial with a custom barrel heat up material and niagara bullet effect added. Enemies were based off a parent enemy blueprint class to encapsulate basic functionality which allowed enemies to share a common behavior tree for AI.
